# NovaDefender AntiCheat

## 利用規約

このアドオンを改造し、二次配布を行うことは禁止されています
自己製作発言もおやめください

## 使用方法

> [!CAUTION]
> **βAPIとeducation editionを有効にしてください** これをしないとアドオンが正常に動作しない場合があります。

### 設定画面を開く

設定画面は;settingで開くことができます
